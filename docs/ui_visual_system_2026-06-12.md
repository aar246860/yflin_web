# UI Visual System

## Direction

Quiet authority. Scientific but not old academic. Collaboration-facing but not SaaS-hype.

## Palette

- Warm paper background
- Dark ink text
- Teal research signal
- Brick decision/risk accent
- Muted gold diagnostic accent
- Field blue for subsurface context

## Components

- Evidence badges
- Concept cards
- Field-note cards
- Research lineage timeline
- Publication clusters
- Collaboration CTA band
- Static search block

## Avoid

- Student-recruitment hero
- Generic AI gradients
- Decorative bokeh/orb visuals
- Grandiose claims
- "World-leading" or "revolutionary" language
