# Research Lineage Map

## Core Line

1. Analytical well hydraulics and boundary physics
   - Pumping tests, Robin boundaries, islands, streams, fault zones, skin effects, slug tests, and heat/solute transport.

2. Lagging theory and groundwater memory
   - The 2017 lagging model becomes a broader delayed-response language for groundwater and thermal signals.

3. Transformation uncertainty
   - TU_Lag reframes aquifer-test interpretation as a transformation from measured drawdown to apparent parameters and decision endpoints.

4. Subsurface energy intelligence
   - TRT, shallow geothermal, aquifer thermal storage, and industrial water-energy planning become the application surface.

## Positioning

The website should present Lin's value as a framework builder: exact-solution discipline, delayed-response modeling, uncertainty auditing, and decision propagation.

## Claim Boundary

Avoid claiming that lagging theory is universally true or that existing groundwater uncertainty methods are obsolete. The stronger claim is that front-end transformation uncertainty should be audited before apparent parameters are used downstream.
