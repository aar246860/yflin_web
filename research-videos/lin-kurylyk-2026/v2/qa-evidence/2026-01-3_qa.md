# Visual QA: periodic-head Lagging Theory

- Reviewer: Codex rendered-frame reviewer
- Review method: visual inspection of 54 exact MP4 frames (entry, midpoint, and settled for all 18 scenes)
- Frame identity: each decision records the zero-based MP4 frame index and SHA-256 of the reviewed image
- Layout decision rule: pass means overlap-free and within-frame; a generic pass label is not accepted
- Evidence record: the visual-QA manifest binds every reviewed frame to the published MP4
- Browser playback requirement: finite duration, positive dimensions, and no media error
- Mobile condition: settled frames reviewed at 390 px width for legible focal geometry, labels, formulas, and boundaries
- Scientific checks: phase increases continuously with distance; the eta=1 boundary is theta=1; field-fit panels distinguish paper observations from digitized published fit curves; numerical validation compares amplitude and phase; the final return resolves to field transects and inferred parameters
- Source policy: the public package contains a copyright-safe evidence digest rather than publisher full text

The signed frame-level decisions and content summaries are recorded in `2026-01-3_visual_qa.json`.
