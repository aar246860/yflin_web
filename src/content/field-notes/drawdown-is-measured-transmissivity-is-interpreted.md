Groundwater professionals often describe a pumping test as providing transmissivity and storage. More precisely, the test provides drawdown or recovery observations. An interpretation model then estimates apparent transmissivity and storage from those observations.

This distinction identifies where uncertainty enters the analysis.

If the interpretation model changes, the apparent parameter set can change. If the parameter set changes, predicted drawdown, allowable pumping, recovery time, and design threshold can change.

Before evaluating downstream uncertainty in a groundwater model, review the transformation from measured drawdown or recovery to interpreted aquifer parameters.
